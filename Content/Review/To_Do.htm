IMDC from David Schnur

IMDC has received a couple of visual changes since 5.5 that might require new screenshots. You can see the full list of changes in the release notes, but the biggest are:

- Reports charts look completely different, and we support viewing them on the iPad.

- Customer-defined datasets now appear in the Reports datasets drop-down with a 'subtitle' indicating which variable & probe they came from.

- The IMDatabase Overview page has been reworked; instead of a bunch of smaller charts it's now just a single estimates chart and a table showing current row counts.

- The IMDatabase Maintenance page has received a copy of the estimates chart and had some of its controls rearranged.

Visual changes aside, there weren't any real functionality changes, so there should be very few actual documentation tweaks. The only thing I can think of offhand is that charts now have a check-box to control whether the legend is visible or hidden.

There might be additional tweaks during the beta period, so I don't know that you should take screenshots immediately, but I figured I'd give you a heads-up, since it's about 95% final right now.
