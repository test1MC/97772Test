User Interface Enhancements

The InterMapper user interface has been redesigned and restructured. It now uses JavaFx, HTML5, and CSS to control the display of status windows and popup windows. Other UI elements, such as dialogs, also use CSS.

  • New Status Window layout. See Status Windows(Pg 1) for more information.
  • New color and shading scheme for dialogs and settings windows. From the InterMapper Preferences window's Colors & Fonts panel, choose from three schemes: Light, Dark, and Classic.

 

System Requirements

System requirements for InterMapper are now maintained only online. Please see the System Requirements page on the HelpSystems website.
